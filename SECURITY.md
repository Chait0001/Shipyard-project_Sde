# Security Policy

## Supported Versions

We actively provide security updates and patches for the following versions of **Shipyard**:

| Version | Supported |
| ------- | --------- |
| v1.0.x  | Yes       |
| < v1.0  | No        |

## Reporting a Vulnerability

We take the security of our application seriously. If you find a security vulnerability, please do **not** report it via a public GitHub issue. Instead, report it responsibly by following these steps:

1.  Send an email to **security@shipyard-app.com** with details of the vulnerability.
2.  Provide a detailed description of the vulnerability, including:
    *   Steps to reproduce the vulnerability (including exploit code or proof of concept if available).
    *   The potential impact of the vulnerability.
    *   Any suggested mitigations.

We will acknowledge receipt of your email within 48 hours and work with you to resolve the issue promptly. We request that you give us reasonable time to fix the vulnerability before making it public.
